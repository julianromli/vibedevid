### 2. Center Navigation Items (New Requirement)
**`components/ui/navbar.tsx` lines 156-224:**

**Problem:** Current layout uses `justify-between` which makes navigation items shift based on left/right content width.

**Solution:** Use a 3-column grid layout where:
- Left column: Logo (flex-1, justify-start)
- Center column: Navigation items (flex-1, justify-center)
- Right column: Actions (flex-1, justify-end)

**Change from:**
```tsx
<nav className="flex h-16 items-center justify-between px-4 md:px-6">
  {/* Left Side */}
  <motion.div className="flex items-center gap-3">...</motion.div>
  
  {/* Center Navigation */}
  <motion.div className="hidden items-center gap-1 md:flex">...</motion.div>
  
  {/* Right Side */}
  <motion.div className="hidden items-center gap-2 md:flex">...</motion.div>
</nav>
```

**Change to:**
```tsx
<nav className="grid h-16 grid-cols-3 items-center px-4 md:px-6">
  {/* Left Side */}
  <motion.div className="flex items-center justify-start gap-3">...</motion.div>
  
  {/* Center Navigation */}
  <motion.div className="hidden items-center justify-center gap-1 md:flex">...</motion.div>
  
  {/* Right Side */}
  <motion.div className="hidden items-center justify-end gap-2 md:flex">...</motion.div>
</nav>
```

This ensures the navigation stays centered even when left/right content varies in width.
